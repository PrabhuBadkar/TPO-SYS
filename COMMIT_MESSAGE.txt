### Backend API
- POST /api/auth/register/recruiter - Full registration
- POST /api/auth/login/recruiter - Login with approval check
- GET /api/auth/recruiter/status - Status tracking
- GET /api/internal/admin/recruiters/all - Get all recruiters
- GET /api/internal/admin/recruiters/pending - Pending list
- GET /api/internal/admin/recruiters/:id - Details
- PUT /api/internal/admin/recruiters/:id/approve - Approve
- PUT /api/internal/admin/recruiters/:id/reject - Reject

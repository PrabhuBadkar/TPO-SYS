## Getting Started

### Prerequisites

- Node.js (v14 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd TPO-SYS
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

### Usage

Run the application:
```bash
npm start
```

Run in development mode:
```bash
npm run dev
```

Run tests:
```bash
npm test
```

## Development

- Add your source code in the `src/` directory
- Add tests in the `tests/` directory
- Add configuration files in the `config/` directory
- Add documentation in the `docs/` directory
